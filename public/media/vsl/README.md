# Interactive VSL media

Add compressed H.264 MP4 files here for the self-hosted VSL flow.

Expected folder structure:

- `intro/founder-intro.mp4`
- `space-types/office.mp4`
- `space-types/home-studio.mp4`
- `space-types/church.mp4`
- `space-types/school.mp4`
- `space-types/restaurant.mp4`
- `space-types/other.mp4`

Optional WebM files can use the same basename with `.webm`.
